- **Sintaxe S3 (imports):** trГӘs formas canГҙnicas вҖ” seletivo `import path (A, B)`, alias `import path = alias` (path Г  esquerda), mГіdulo inteiro `import path` (sГі caminho completo, sem alias implГӯcito do Гәltimo segmento). - **Sintaxe S3 (P0 вҖ” superfГӯcie de arquivo):** cabeГ§alho canГҙnico `module path` (substitui `namespace`; `namespace` вҶ’ `parse.namespace_removed`).
