## Notes / fairness

- Same algorithm shape (while loops, explicit indices) across languages.
- Rust: `cargo build --release` without fat LTO; `black_box` on final result so the value is not DCE'd.
- Python: CPython 3, no PyPy / no numba.
- Ori: AOT binary via `ori compile` (not JIT `ori run`).
- Times include process start + one-line stdout.
- Fib is **N iterative steps**, not classic F(40) (too small to measure).
- This is **not** a language ranking for all domains — only these microkernels.
